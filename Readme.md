## 安装
```bash
npm i -g @gm5/cli
```

## 用法
> 用法: `five-cli [command] args...`

+ Commands:
  * init             -    初始化框架
  * start [prod|dev] -    运行当前的应用(可指定运行环境)
  * stop             -    停止当前应用
  * st|status        -    查看当前应用状态
  * r|restart        -    重启当前应用
  * del|delete       -    删除当前应用
  * -h               -    查看帮助文档
  * -v               -    查看工具的版本
